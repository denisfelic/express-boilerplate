# express-boilerplate

- Express
- Typescript
- TypeORM
- SQLITE3


```
npm install
```

Copy .env.example to .env and configure port
```
cp .env.example .env
```

Start dev
```
npm run dev
```
Build for production
```
npm run build
```
Start production 
```
npm run start
```

Default test route `/api/v1/users`
